Usage

Make HTTP GET request to /api/timestamp to get current time as UTC time string and UNIX epoch time value in JSON

Make HTTP GET request with route param:

  • Unix/Epoch time value in 10-digit (seconds) format, e.g. /api/timestamp/1500000000
  • Date string: "2020-01-01", "2020-Jan-01", RFC 3339
